## 📌 Description
This project implements a touchless Spotify controller using Computer Vision and MediaPipe.

The system detects hand gestures using webcam input and maps them to Spotify controls like:
- Play / Pause
- Next / Previous track
- Volume Control
- Skip forward

## 🛠️ Tech Stack
- Python
- OpenCV
- MediaPipe
- AppleScript (macOS)

## 🎯 Features
- Real-time hand tracking (21 landmarks)
- Gesture recognition
- Cooldown system to prevent repeated triggers
- Visual feedback using OpenCV
